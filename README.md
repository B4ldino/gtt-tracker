## 📝 Descrizione
Questa applicazione è stata creata per semplificare la ricerca delle fermate, offrendo un'interfaccia pulita, veloce e accessibile da qualsiasi dispositivo. Non serve più perdere tempo a cercare tra mille pagine: basta un input e l'app ti restituisce le informazioni che ti servono.
